# CARIFet (Cabinet of the Republic of Indonesia Fetcher)
Fetch information of current Indonesian Cabinet from Wikipedia.

Dependencies

* Pandas
* Matplotlib
* Wikipedia-API
* wikipedia
* BeautifulSoup
* googlesearch
